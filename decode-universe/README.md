## 🎯 What is DECODE?

DECODE is a **transmedia MIL education ecosystem** with 3 interconnected pillars:

| Pillar | Name | Role |
|--------|------|------|
| 📺 Video | DECODE Stories | **HOOK** — Sparks curiosity with real stories |
| 🎲 Game | DECODE: The Game | **EXPERIENCE** — Learn by investigating content |
| 📖 Comics | DECODE Chronicles | **DEEP DIVE** — Empathy through character stories |

## 🔧 The DECODE Method

6 steps to decode any information:

1. **D**etect — Spot red flags
2. **E**vidence — Check for proof
3. **C**ross-check — Verify with multiple sources
4. **O**rigin — Investigate the creator
5. **D**etermine motive — Understand intent
6. **E**valuate impact — Consider consequences
